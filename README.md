# Image Resizer

이미지를 드래그해서 업로드하고, 크기를 조절한 후 다운로드할 수 있는 애플리케이션입니다.

## 프로젝트 구조

```
image-resizer/
├── client/          # React 프론트엔드
├── server/          # Express 백엔드
└── README.md
```

## 주요 기능

- 🖼️ 드래그 앤 드롭으로 이미지 업로드
- 📦 여러 이미지 한 번에 업로드
- 🎨 이미지 크기 조절
- ↔️ 원본 비율 유지
- 💾 일괄 다운로드
